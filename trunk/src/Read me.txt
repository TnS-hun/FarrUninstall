General
~~~~~~~
FarrUninstall lets you uninstall programs from Find and Run Robot. It is much faster than Add or Remove programs. :)
This program is free for both personal and commercial use.
Source code included.

Installation
~~~~~~~~~~~~
To install simply copy the whole directory to the plugins directory of Find and Run Robot. (Eg. C:\Program Files\FindAndRunRobot\Plugins\)

Usage
~~~~~
un keywords -- search uninstall list
un ` -- reload uninstall list
